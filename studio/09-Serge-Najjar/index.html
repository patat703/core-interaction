BETH COMSTOCK BRANDING

LETA SOBIERAJSKI

Beth Comstock Branding is a project created by Leta Sobierajski and her co-worker, Wade Jeffree, George Stoyanov, and Scottie Cameron. The project was created for Beth Comstock, who aims to develop the study of how humans will interact with technology in the near future, and how humans will learn from machines as machines will learn from humans. The purpose of this project is to create a branding solution for Beth that would embody her ideas of "change, evolution, innovation, and emergence".
